# SRTP Final Project

本仓库用于整理 SRTP 结项阶段围绕 myCobot-280 与 ROS2 `JointState` 链路开展的攻击复现、规则推理建模、防护设计与实验记录。

## 仓库结构

- `report/`
  - 结项报告正文草稿与后续正式稿。
- `Saure_work/`
  - 详细实验记录、分析笔记、XSB 规则文件与阶段性材料。
- `reference/`
  - 开题与中期阶段参考材料、早期 XSB 规则与示例代码。
- `code/`
  - 结项阶段实际相关的 ROS2 防护代码快照与说明。
- `Saure.md`
  - 项目推进计划与阶段复盘记录。
- `draft.md`
  - 临时草稿、命令输出与中间整理内容。

## 研究主线

本项目当前聚焦于以下闭环：

1. 在 ROS2 虚拟环境中复现基于 `JointState` 的显示层状态欺骗；
2. 复现合法 publisher 与恶意 publisher 并存导致的竞争性状态振荡；
3. 通过 XSB / MulVAL 风格规则推理解释攻击链成立条件；
4. 设计防护版规则模型并验证攻击链失效；
5. 在 ROS2 代码层实现最小应用层防护，并完成运行时验证。

## 代码说明

`code/` 目录不直接保存整个上游 `mycobot_ros2` 工作区，而是仅保留本项目结项阶段真正相关的防护代码快照。这样做的原因是：

- 避免提交整个 `colcon_ws` 及其构建产物；
- 避免将上游仓库的嵌套 `.git` 一并带入当前项目仓库；
- 便于在报告与答辩中直接定位本项目新增或修改的关键文件。
